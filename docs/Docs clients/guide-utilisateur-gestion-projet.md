# Sommaire

# 1. Introduction — à quoi sert ce module ?
Le module Gestion de projet de Pladigit vous permet de piloter vos projets municipaux de A à Z : suivi des tâches, planning, budget, risques, communication avec les élus. C’est l’équivalent libre et souverain de Microsoft Planner.

# 2. Les rôles dans un projet
Un projet Pladigit distingue deux niveaux de droits : votre rôle global dans l’organisation (Admin, DGS, Resp. Service…) ET votre rôle dans le projet spécifique. Les deux se cumulent.




# 3. Créer un nouveau projet
## 3.1 — Procédure
1. Cliquez sur « Projets » dans le menu de gauche.
2. Cliquez sur le bouton « + Nouveau projet » en haut à droite.
3. Remplissez le formulaire :
- Nom du projet (obligatoire)
- Description (optionnelle — supporte le texte mis en forme)
- Dates de début et de fin prévue
- Couleur (pour le distinguer dans les listes)
- Budget total prévisionnel (optionnel)
- Visibilité : Normal (visible par la hiérarchie) ou 🔒 Privé (membres uniquement)
4. Cliquez sur « Créer le projet ». Vous êtes automatiquement désigné Chef de projet.

## 3.2 — Ajouter des membres
1. Dans la page du projet, cliquez sur « Modifier » puis sur l’onglet « Membres ».
2. Sélectionnez un utilisateur dans la liste déroulante.
3. Choisissez son rôle : Contributeur ou Observateur.
4. Cliquez sur « Ajouter ».


# 4. Organiser son projet — Phases et Jalons
## 4.1 — Comprendre la structure
Un projet Pladigit s’organise en :
- 📌 Phases — grands blocs de travail (ex : « Phase 1 — Étude de faisabilité »). Une phase contient des jalons.
- 🏁 Jalons — étapes clés avec une date d’échéance (ex : « Livraison du rapport »). Un jalon contient des tâches.
- ✔ Tâches — actions concrètes à réaliser, assignées à un membre.

## 4.2 — Créer une phase
1. Dans la section « But & description » du projet, cliquez sur « + Nouvelle phase ».
2. Donnez un titre à la phase et une date de fin prévue.
3. Choisissez une couleur pour distinguer la phase visuellement dans le Gantt.
4. Cliquez sur « Enregistrer ».
## 4.3 — Créer un jalon
1. Cliquez sur « + Nouveau jalon » sous la phase souhaitée (ou en autonome).
2. Renseignez le titre et la date d’échéance (obligatoire).
3. La date de début est optionnelle — utile pour l’affichage Gantt.
## 4.4 — Marquer un jalon comme atteint
1. Sur le jalon concerné, cliquez sur le bouton vert « ✓ Atteint ».
2. Confirmez. Le jalon passe en vert barré.
3. Vous pouvez ajouter un commentaire (note de clôture visible par les élus).

## 4.5 — Réordonner les phases et jalons
Utilisez les flèches ↑ et ↓ affichées à côté de chaque phase/jalon pour modifier leur ordre d’affichage.

# 5. Gérer les tâches
## 5.1 — Créer une tâche
1. Dans la vue Liste ou Kanban, cliquez sur « + Nouvelle tâche ».
2. Remplissez le panneau latéral :
- Titre (obligatoire)
- Statut : À faire / En cours / En revue / Terminé
- Priorité : Basse / Moyenne / Haute / Urgente
- Assigné à : membre du projet
- Jalon d’appartenance
- Dates de début et d’échéance (utiles pour le Gantt)
- Heures estimées
3. Cliquez sur « Enregistrer ».

## 5.2 — Avancer une tâche
1. Cliquez sur la tâche dans la liste ou la carte Kanban.
2. Modifiez le statut dans le panneau latéral : sélectionnez « En cours », « En revue » ou « Terminé ».
3. Renseignez les heures réalisées pour mettre à jour la charge.

## 5.3 — Dépendances entre tâches
Certaines tâches ne peuvent pas démarrer tant qu’une autre n’est pas terminée. Pour ajouter une dépendance :
1. Ouvrez la tâche concernée.
2. Dans la section « 🔗 Dépendances », cliquez sur « + Ajouter ».
3. Sélectionnez la tâche prédécesseure dans la liste.

## 5.4 — Tâches récurrentes
Pour une tâche qui se répète (réunion hebdomadaire, rapport mensuel) :
1. Dans le panneau de la tâche, section « Récurrence ».
2. Choisissez le type : Quotidien, Hebdomadaire ou Mensuel.
3. Indiquez la fréquence (tous les N jours/semaines/mois) et une date de fin optionnelle.

## 5.5 — Commenter une tâche
En bas du panneau de tâche, saisissez votre message dans le champ « Ajouter un commentaire » et cliquez sur « Envoyer ». Tous les membres du projet voient les commentaires.
## 5.6 — Joindre un document à une tâche
1. Ouvrez la tâche.
2. Dans la section « 📎 Documents », cliquez sur « 📎 Fichier » ou « 🔗 Lien ».
3. Sélectionnez le fichier (PDF, Word, Excel, image… max 50 Mo) ou saisissez l’URL.

# 6. Les vues de planification
L’onglet « Tâches & planning » propose 5 façons de visualiser vos tâches : Liste, Kanban, Gantt, Agenda et Charge.
## 6.1 — Vue Liste
Un tableau clair groupé par jalon. Par défaut, seules les tâches en cours sont affichées. Cochez « Toutes les tâches » pour tout voir.
- Cliquez sur un en-tête de colonne (Nom, Priorité, Assigné, Échéance) pour trier.
- Utilisez la barre de recherche pour filtrer par mot-clé.
- Le sélecteur de tri en haut permet de trier aussi le Kanban.
## 6.2 — Vue Kanban
4 colonnes : À faire / En cours / En revue / Terminé. Les tâches sont groupées par jalon.
- Glissez une carte d’une colonne à l’autre pour changer son statut.
- Cliquez sur le bouton « + Ajouter » en bas d’une colonne pour créer une tâche directement dans ce jalon.
- Les jalons atteints sont repliés automatiquement.

## 6.3 — Vue Gantt
Un diagramme de barres pour visualiser les dates et la progression de vos tâches dans le temps.
- Utilisez les boutons + et − ou la molette pour zoomer.
- Cliquez sur l’en-tête d’un jalon pour le déplier/replier.
- 🆕 Glissez une barre horizontalement pour décaler les dates de la tâche. La modification est enregistrée automatiquement.
- Un clic simple sur une barre ouvre le détail de la tâche.

## 6.4 — Vue Agenda
Liste chronologique des événements liés au projet : réunions, ateliers, auditions. Un clic sur un événement en affiche le détail. Vous pouvez créer, modifier et supprimer des événements selon vos droits.
- Visibilité 🕒 Privé : visible uniquement par vous.
- Visibilité 🔵 Restreint : visible par tous les membres du projet (valeur par défaut).
- Visibilité 🟢 Public : visible par tous les utilisateurs de l’organisation.
## 6.5 — Vue Charge
Affiche la répartition des heures de travail par membre sur la période. Utile pour éviter la surcharge.
- Bleu : charge normale.
- Jaune : chargé (3+ tâches dans la semaine).
- Rouge : surchargé (5+ tâches).
- Cochez « Toutes les tâches » pour voir aussi les tâches à faire et terminées.

# 8. Conduite du changement
## 8.1 — Parties prenantes
La matrice des parties prenantes recense toutes les personnes ou entités impactées par le projet et leur niveau d’adhésion.
- Champion : promoteur actif du projet.
- Supporteur : favorable, sans s’impliquer activement.
- Neutre : sans opinion marquée.
- Résistant : réticent — à convaincre.
- Opposé : activement contre — risque à gérer.

## 8.2 — Freins & risques
La matrice des risques classe chaque risque selon sa probabilité (Faible / Moyenne / Forte) et son impact (Faible / Moyen / Fort / Critique). La criticité est calculée automatiquement.
1. Cliquez sur « + Ajouter un risque » dans la section Freins & risques.
2. Renseignez : description, catégorie, probabilité, impact, plan de mitigation.
3. Suivez le statut : Identifié → En cours de traitement → Clôturé.
## 8.3 — Plan de communication
La timeline de communication liste toutes les actions de communication prévues (réunions d’information, affichages, courriers). Chaque action a un canal, un responsable et une date planifiée.

# 9. Tableau de bord élus
L’onglet « Tableau de bord élus » offre une vision synthétique du projet destinée aux élus et à la direction : KPIs, alertes, avancement budget, jalons, risques actifs.
## 9.1 — Exporter en PDF
1. Dans l’onglet Tableau de bord élus, cliquez sur « 📄 Exporter en PDF ».
2. Le fichier se télécharge immédiatement.
## 9.2 — Exporter en ZIP (PDF + CSV)
1. Cliquez sur « 🗃 Exporter en ZIP ».
2. L’archive contient : le PDF du tableau de bord, les tâches en CSV, le budget en CSV, les risques en CSV.
## 9.3 — Exporter les jalons en iCal
1. Cliquez sur le bouton « Jalons .ics ».
2. Importez le fichier dans Outlook, Google Agenda ou tout autre agenda compatible.


# 10. Documents et pièces jointes
Vous pouvez joindre des fichiers ou des liens URL directement sur le projet ou sur une tâche.
## 10.1 — Joindre un document au projet
1. Dans le menu latéral du projet, cliquez sur « 📄 Documents ».
2. Cliquez sur « 📎 Joindre un fichier » ou « 🔗 Ajouter un lien ».
3. Formats acceptés : PDF, Word (.doc, .docx), Excel (.xls, .xlsx), PowerPoint, ODF, images, ZIP. Taille max : 50 Mo.
## 10.2 — Télécharger un document
Cliquez sur « ⬇ Télécharger » à côté du document. Pour un lien, cliquez sur « Ouvrir ».


# 11. Agenda et événements
## 11.1 — Créer un événement
1. Dans l’onglet Agenda, cliquez sur « + Nouvel événement ».
2. Renseignez : titre, date/heure de début et de fin, lieu, description, visibilité.
3. Si une visioconférence est prévue, cochez « Créer un lien de visio ». Un lien Jitsi Meet est généré automatiquement.
## 11.2 — Rejoindre une visioconférence
1. Cliquez sur le bouton 🌐 dans la barre latérale du projet ou dans l’événement.
2. Une fenêtre s’ouvre avec l’URL de la réunion. Cliquez sur « Rejoindre ».

# 13. Historique d’activité
Le bouton « Historique » dans la barre latérale affiche le journal chronologique de toutes les actions sur le projet : création de tâches, changements de statut, ajouts de membres, jalons atteints…
- Filtrez par type d’action grâce au sélecteur en haut.
- Les membres voient leurs propres actions. Les chefs de projet et la direction voient tout l’historique.
- L’historique est chargé au premier clic (pas à chaque ouverture de page).

# 14. Modèles de projets
Les modèles permettent de capitaliser sur des structures de projets récurrents (ex : « Organisation d’une fête municipale », « Projet de voirie »).
## 14.1 — Créer un modèle depuis un projet existant
1. Dans la page du projet, cliquez sur « Modifier » → onglet « Actions ».
2. Cliquez sur « Sauvegarder comme modèle ».
3. Donnez un nom au modèle. Les phases, jalons et tâches types sont capturés avec leurs décalages en jours.
## 14.2 — Créer un projet depuis un modèle
1. Sur la page Projets, cliquez sur « + Nouveau projet ».
2. Sélectionnez « Partir d’un modèle ».
3. Choisissez le modèle et renseignez la date de démarrage. Toutes les dates sont calculées automatiquement.
## 14.3 — Dupliquer un projet existant
1. Dans « Modifier » → onglet « Actions », cliquez sur « Dupliquer ce projet ».
2. Indiquez une nouvelle date de démarrage. Les tâches et jalons sont recopiés avec les dates décalées.
