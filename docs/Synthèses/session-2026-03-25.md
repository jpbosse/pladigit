## Task 4 — Enforcement strict du quota

### MediaService
- `quotaInfo()` : source de vérité unique (`usedBytes`, `quotaBytes`)
- `assertQuota()` refactorisé pour utiliser `quotaInfo()`
- `notifyQuotaThresholdIfCrossed(int $addedBytes)` : alerte automatique à 80 / 90 / 95%
- `ingestNasFile()` retourne `bool` (false = quota dépassé, fichier ignoré)
- Compteur `$skipped` incrémenté dans `syncByMtime()` quand quota dépassé

### NotificationService
- `quotaWarning(int $usedBytes, int $quotaBytes, int $threshold)` : notification in-app
  aux utilisateurs `role = admin` avec type `storage.quota_warning`
